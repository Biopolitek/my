Техническое Задание: Project Chimera (v2.4.4 Production Build)
Статус: Утверждено (Финальная версия).
Исполнитель: AI-команда разработки Chimera Solutions.
Дата: 30 января 2026 г.
1. Цель проекта и Обзор (Team Lead Summary)
Создание автономного программного комплекса для высокоточной локальной конвертации дизайн-макетов в функциональный веб-интерфейс. Система гарантирует 100% приватность данных и обеспечивает автоматическую генерацию кликабельных ссылок, карт сайта и структуры ресурсов.
2. Стек Технологий (Full-stack Developer Scope)
Компонент	Технология	Обоснование
Ядро	Python 3.12+ (LTS)	Стабильность и поддержка современных библиотек.
Интерфейс	PySide6 (Qt)	Профессиональный UI, работа в потоках QThread.
Парсинг PSD	psd-tools	Глубокое извлечение слоев и метаданных.
Парсинг PDF	pdfminer.six	Векторное извлечение текста и координат.
Парсинг FIG	zipfile + binary	Доступ к бинарному ядру canvas.fig.
Рендеринг	Jinja2	Генерация адаптивного HTML-кода по шаблонам.
Мониторинг	PowerShell CIM	Прямое считывание температуры и ресурсов (UAC-Admin).
3. Функциональные требования (Designer & QA Scope)
3.1. Обработка и Экспорт (100% Persistence)
Layer Extraction: Физическое извлечение всех видимых слоев из PSD/FIG и сохранение в ./Export/assets/.
Typography Engine: Автоматическое копирование файлов шрифтов из internal_assets и генерация fonts.css с правилами @font-face.
SEO Pack: Автоматическая генерация sitemap.xml и robots.txt на основе проанализированного графа страниц.
3.2. Интеллектуальный анализ (Semantic Anchor)
Граф связей: Алгоритм сопоставления текстовых блоков и имен слоев для создания активных ссылок (например, «Contact» -> contacts.html).
Jinja2 Render Service: Формирование HTML-страниц с поддержкой Mobile/Desktop режимов (адаптивный вьюпорт и сетка).
3.3. Интерфейс и Мониторинг (UI Scope)
Визуальный контроль: 3-х зонный интерфейс (Мониторинг, Таблица анализа, Лог событий).
Dynamic ETA: Отображение оставшегося времени в формате ММ:СС, рассчитанное на основе веса (MB) файлов.
Thermal Guard: Изменение фона статус-бара на красный (#c0392b) и вывод предупреждения «ПАУЗА: ПЕРЕГРЕВ» при нагрузке CPU > 85%.
4. Архитектура и Отказоустойчивость
Изоляция проектов: Принудительная очистка папки Export перед началом каждого нового цикла конвертации.
Admin Mode: Приложение должно запрашивать права Администратора (UAC) для корректной работы аппаратного мониторинга.
Stable Pathing: Использование resource_path для корректной работы путей ассетов внутри скомпилированного EXE-пакета.
5. Поставка и Сборка
Формат: Standalone EXE (режим --onedir).
Команда сборки:
python -m PyInstaller --noconfirm --onedir --windowed --uac-admin --name "Chimera_v2.4.4_Final" --collect-all jinja2 --hidden-import "psd_tools" --add-data "internal_assets;internal_assets" "main_app.py"
